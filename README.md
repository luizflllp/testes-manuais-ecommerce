# Projeto de Testes Manuais – E-commerce

Este projeto tem como objetivo demonstrar minhas habilidades em testes manuais, incluindo criação de casos de teste e reporte de bugs.

## 🔍 Escopo
Testes realizados em funcionalidades principais:
- Login
- Cadastro
- Carrinho de compras

## 🛠️ Ferramentas
- Testes manuais
- Postman (API)
- Documentação de bugs

## 📌 Objetivo
Praticar e demonstrar conhecimentos em QA para oportunidades como QA Júnior.
