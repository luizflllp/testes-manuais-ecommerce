# Casos de Teste

## CT01 - Login válido
**Cenário:** Usuário faz login com dados corretos  
**Passos:**
1. Acessar página de login  
2. Inserir email válido  
3. Inserir senha válida  
4. Clicar em "Entrar"  

**Resultado esperado:**  
Usuário deve acessar o sistema com sucesso  

---

## CT02 - Login inválido
**Cenário:** Usuário tenta login com senha incorreta  

**Resultado esperado:**  
Sistema deve exibir mensagem de erro  

---

## CT03 - Adicionar produto ao carrinho
**Cenário:** Usuário adiciona produto ao carrinho  

**Resultado esperado:**  
Produto deve ser adicionado corretamente
